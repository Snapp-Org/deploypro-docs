# GCP - Short Introduction 

Google Cloud Platform (GCP) is a cutting-edge public cloud vendor, offering a suite of high-end computing services. From data management to web and video content delivery, and advanced AI and machine learning tools, GCP caters to diverse needs. With its reliable and scalable infrastructure, seamless application deployment and management become effortless. 

## Overview
In this Google Clous PLatform (GCP) topic, you will find tutorial about:

- Login to Google Cloud Platform
- Add a user to create an access key and secret key on AWS
- Log in to DeployPRO
- Add your AWS keys to your DeployPRO Account
- Deploy a static app in DeployPRO
