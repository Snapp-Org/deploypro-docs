# Script Languages

This page contains a short list with script languages currently used in Cloud Programming:

> **Bash**

A shell scripting language commonly used for automation and running command-line tasks. It's especially useful for managing Linux-based cloud environments.

> **Python**

Widely used for scripting, automation, and building cloud applications. It has a rich ecosystem of libraries and frameworks that make it versatile for various cloud-related tasks.

> **Perl** 

A versatile scripting language used for automation, text processing, and system administration. It's been used for various cloud-related tasks, although its usage has decreased over time.

> **PowerShell**

Microsoft's scripting language primarily used for managing Windows environments and interacting with Microsoft Azure services. PowerShell Core is cross-platform and can be used in non-Windows environments as well.

> **Ruby**

Known for its simplicity and elegance, Ruby is used for scripting, automation, and web development. It's often used with configuration management tools like Chef and Puppet.

> **Lua** 

A lightweight scripting language known for its embeddability. It's used in some cloud-related tools and frameworks for configuration and scripting.
