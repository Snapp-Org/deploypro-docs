# GCP - Glossary

**Google Cloud Platform** (GCP) by `Google` terms explained - Compute Engine, Cloud Firestore, Cloud Spanner and much more.

## A 

**AI Platform**

    Machine learning platform that provides tools for building, training, and deploying machine learning models.

**Anthos**

    A platform for managing applications across multiple clouds and on-premises environments.

## B 

**BigQuery**

    A fully managed data warehouse service for analyzing large datasets using SQL-like queries.

## C 

**Cloud Storage**

    Scalable object storage for storing and retrieving any amount of data from anywhere on the web.

**Cloud Run**

    A fully managed compute platform that automatically scales your stateless containers.

**CDN** 

    Content delivery network service that improves the delivery of web and video content to users.

## D 

**Deployment Manager** 

    Infrastructure-as-code service that allows you to create, manage, and deploy resources using configuration files.

## F

**Functions**

    Serverless compute service similar to AWS Lambda, enabling you to run event-driven code without managing servers.

**Firestore**

    GCP's NoSQL database service for storing, syncing, and querying data for web, mobile, and server development.

## I 

**IAM** 

    Identity and Access Management (IAM): GCP's service for managing permissions and access to resources, controlling who can do what within GCP.

## S

**Spanner**

    Cloud Spanner: Globally distributed and horizontally scalable relational database service.

**Pub/Sub**

    Cloud Pub/Sub: Messaging service for building event-driven systems and real-time analytics.

**SQL**

    Cloud SQL: Managed relational database service for MySQL, PostgreSQL, and SQL Server.

## V 

**VPC**

    Isolated network environments within GCP that allow you to control communication between resources and configure network settings.
