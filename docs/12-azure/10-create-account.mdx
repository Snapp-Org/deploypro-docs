# Azure - Create Account

To connect your Azure account to DeployPRO, first create an Azure account. If you already have an account, generate a credential in Azure.

Here are the steps on how to create an account on Azure:

1. Go to the Azure website: https://azure.microsoft.com/.
2. Click on the `Free Azure Account` button on the top right corner.
3. Click on the `Start Free `button.
4. Sign in or create a Microsoft account.
5. Enter your country/region and date of birth.
6. Click on the `Create Account` button.
